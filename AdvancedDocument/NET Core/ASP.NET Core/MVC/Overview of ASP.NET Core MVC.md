## Features

ASP.NET Core MVC includes the following features:


* Routing

* Model binding

* Model validation

* Dependency injection

* Filters

* Areas

* Web APIs

* Testability

* Razor view engine

* Strongly typed views

* Tag Helpers

* View Components
