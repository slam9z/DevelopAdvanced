[How to display a list inline using Twitter's Bootstrap](http://stackoverflow.com/questions/10906341/how-to-display-a-list-inline-using-twitters-bootstrap)

Bootstrap 3

Use the list-inline class.

```html
<ul class="list-inline">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```
