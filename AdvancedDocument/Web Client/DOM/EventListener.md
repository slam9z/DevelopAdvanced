##Methods

###handleEvent()

This method is called whenever an event occurs of the type for which the EventListener 
interface was registered.

```js
void handleEvent(
  in Event event
);
```
      
##Parameters

###event

The DOM [Event](https://developer.mozilla.org/en-US/docs/Web/API/Event) to register.
